الجزء الأول: Grid مقابل Flexbox — متى تستخدم أيهما؟

1.1

الفرق الجوهري

Flexbox أحادي البُعد: يُنظّم العناصر في صف واحد أو عمود واحد فقط. CSS Grid ثنائي الأبعاد: يُنظّم العناصر في صفوف وأعمدة في آنٍ واحد.

📐 Flexbox (1D) مقابل CSS Grid (2D)
  Flexbox — صف واحد:
  ┌────────────────────────────────────────┐
  │  [Item 1]  [Item 2]  [Item 3]  [Item 4]│  ← محور واحد فقط
  └────────────────────────────────────────┘
  CSS Grid — صفوف وأعمدة معاً:
  ┌──────────┬──────────┬──────────┐
  │  Item 1  │  Item 2  │  Item 3  │  ← صف 1
  ├──────────┼──────────┼──────────┤
  │  Item 4  │  Item 5  │  Item 6  │  ← صف 2
  ├──────────┼──────────┼──────────┤
  │  Item 7  │  Item 8  │  Item 9  │  ← صف 3
  └──────────┴──────────┴──────────┘
     عمود 1    عمود 2    عمود 3
استخدم Flexbox عندما:استخدم CSS Grid عندما:
ترتيب عناصر في صف أو عمود واحدبناء تخطيط الصفحة الكاملة
شريط التنقل (Navbar)شبكة بطاقات بصفوف وأعمدة
مجموعة أزرار أو أيقوناتتخطيط Dashboard معقد
توسيط عنصر داخل حاويةتخطيط مجلة أو صحيفة
توزيع مرن في اتجاه واحدموضع دقيق لكل عنصر في شبكة
عناصر بأحجام غير منتظمةتمديد عنصر عبر عدة أعمدة وصفوف
💡 الحقيقة: Flexbox وGrid ليسا متنافسَين!

المشاريع الاحترافية تستخدم الاثنين معاً:

✦ Grid للتخطيط الخارجي للصفحة: header, sidebar, main, footer

✦ Flexbox للتخطيط الداخلي لكل قسم: محاذاة العناصر داخل header مثلاً

✦ القاعدة: Grid من الخارج للداخل، Flexbox من الداخل للخارج

الجزء الثاني: أساسيات CSS Grid

2.1

تفعيل Grid وتعريف الأعمدة

CSS 
/* تفعيل Grid على الحاوية */
.container {
  display: grid;
}

/* تعريف 3 أعمدة بعرض ثابت */
.container {
  display: grid;
  grid-template-columns: 200px 200px 200px;
}

/* تعريف 3 أعمدة بنسب مئوية */
.container {
  display: grid;
  grid-template-columns: 25% 50% 25%;
}

/* تعريف صفوف بارتفاع محدد */
.container {
  display: grid;
  grid-template-columns: 200px 200px 200px;
  grid-template-rows: 100px 200px 100px;
}
2.2

وحدة fr (Fraction)

وحدة fr تعني «كسر من المساحة المتبقية» — هي الوحدة الأقوى والأكثر استخداماً في Grid:

CSS 
/* 3 أعمدة بمساحة متساوية */
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  /* = كل عمود يأخذ ثلث المساحة المتاحة */
}

/* توزيع غير متساوٍ */
.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  /* sidebar | main | sidebar */
  /* 25%    | 50%  | 25%     */
}

/* مزج بين ثابت ومرن */
.container {
  display: grid;
  grid-template-columns: 280px 1fr;
  /* sidebar ثابت 280px | main يأخذ الباقي */
}

/* 3 أعمدة + footer يمتد لكل العرض */
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
}
📏 توضيح وحدة fr
  grid-template-columns: 1fr 1fr 1fr
  ┌──────────────┬──────────────┬──────────────┐
  │     1fr      │     1fr      │     1fr      │
  │   (33.3%)    │   (33.3%)    │   (33.3%)    │
  └──────────────┴──────────────┴──────────────┘
  grid-template-columns: 1fr 2fr 1fr
  ┌──────────┬────────────────────┬──────────┐
  │   1fr    │        2fr         │   1fr    │
  │  (25%)   │       (50%)        │  (25%)   │
  └──────────┴────────────────────┴──────────┘
  grid-template-columns: 280px 1fr
  ┌────────────┬──────────────────────────────┐
  │   280px    │       1fr (الباقي كله)       │
  │  (ثابت)   │          (مرن)               │
  └────────────┴──────────────────────────────┘
2.3

دالة repeat()

repeat() تختصر تكرار نفس التعريف عدة مرات:

CSS 
/* بدون repeat — مكرر */
grid-template-columns: 1fr 1fr 1fr 1fr;

/* مع repeat — أنيق ومختصر */
grid-template-columns: repeat(4, 1fr);

/* repeat بأعمدة مختلفة */
grid-template-columns: repeat(3, 200px);
/* = 200px 200px 200px */

/* مزج repeat مع قيم ثابتة */
grid-template-columns: 280px repeat(3, 1fr);
/* = 280px 1fr 1fr 1fr */

/* تعريف صفوف بـ repeat */
grid-template-rows: repeat(4, 150px);

/* repeat مع auto — الصف يتمدد حسب المحتوى */
grid-template-rows: repeat(3, auto);
2.4

دالة minmax()

minmax() تحدد حداً أدنى وحداً أقصى لعرض العمود أو ارتفاع الصف — مفتاح التجاوب الحقيقي:

CSS 
/* عمود لا يقل عن 200px ولا يزيد عن 1fr */
grid-template-columns: repeat(3, minmax(200px, 1fr));

/* الوصفة الذهبية لشبكة متجاوبة بدون Media Queries! */
.responsive-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
  gap: 24px;
}
/* auto-fit: يضع أكبر عدد ممكن من الأعمدة */
/* minmax(280px, 1fr): كل عمود بين 280px و1fr */
/* النتيجة: تلقائياً 3 أعمدة → 2 → 1 حسب العرض */

/* auto-fill vs auto-fit */
/* auto-fill: يملأ بأعمدة فارغة إذا لزم الأمر */
/* auto-fit: يُمدّد العناصر لملء المساحة الفارغة ✅ */
grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
⭐ الوصفة الذهبية — احفظها!

repeat(auto-fit, minmax(MIN, 1fr))

هذه الوصفة تُنشئ شبكة متجاوبة تلقائياً بدون أي Media Query!

عندما تضيق الشاشة: تنتقل من 3 أعمدة → 2 أعمدة → 1 عمود تلقائياً

استبدل MIN بأصغر عرض مناسب لبطاقاتك (مثال: 250px, 300px, 350px)

الجزء الثالث: وضع العناصر وتمديدها

3.1

خطوط الشبكة (Grid Lines)

Grid تعمل على خطوط وهمية تُرقَّم تلقائياً. شبكة بـ 3 أعمدة لها 4 خطوط عمودية (1 إلى 4) وشبكة بـ 2 صف لها 3 خطوط أفقية (1 إلى 3):

🔢 أرقام خطوط الشبكة (Grid Lines)
       1        2        3        4     ← خطوط الأعمدة
  1    ┌────────┬────────┬────────┐
       │ [1,1]  │ [2,1]  │ [3,1]  │
  2    ├────────┼────────┼────────┤
       │ [1,2]  │ [2,2]  │ [3,2]  │
  3    ├────────┼────────┼────────┤
       │ [1,3]  │ [2,3]  │ [3,3]  │
  4    └────────┴────────┴────────┘
  ↑
  خطوط الصفوف
  [عمود, صف] — كل خلية بين خطين عموديين وخطين أفقيين
3.2

grid-column وgrid-row

CSS 
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: repeat(3, 150px);
  gap: 16px;
}

/* وضع عنصر في عمود محدد */
.item-a {
  grid-column: 2;    /* يبدأ من الخط 2 (العمود الثاني) */
  grid-row: 1;       /* في الصف الأول */
}

/* تمديد عنصر عبر أعمدة — من الخط 1 إلى الخط 3 */
.item-header {
  grid-column: 1 / 3;    /* يمتد من الخط 1 حتى الخط 3 (عمودان) */
  grid-row: 1 / 2;       /* الصف الأول فقط */
}

/* تمديد عنصر بكلمة span */
.item-wide {
  grid-column: span 2;   /* يمتد بعرض عمودين */
  grid-row: span 2;      /* يمتد بارتفاع صفين */
}

/* -1 تعني آخر خط (نهاية الشبكة) */
.item-full-width {
  grid-column: 1 / -1;   /* يمتد من أول خط لآخر خط = عرض كامل */
}
CSS 
/* مثال عملي: تخطيط magazine */
/*
  ┌─────────────────────────────┐
  │       Featured Article      │  ← يمتد 2 عمود + 2 صف
  │         (2col × 2row)       │
  ├──────────────┬──────────────┤
  │   Article 2  │   Article 3  │  ← عمود × صف
  └──────────────┴──────────────┘
*/
.magazine-grid {
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  grid-template-rows: repeat(3, 200px);
  gap: 20px;
}

.featured {
  grid-column: 1 / -1;   /* عرض كامل */
  grid-row: 1 / 3;       /* صفان */
}

.article-2 {
  grid-column: 1;
  grid-row: 3;
}

.article-3 {
  grid-column: 2;
  grid-row: 3;
}

الجزء الرابع: مناطق الشبكة (Grid Template Areas)

grid-template-areas تُتيح تسمية مناطق الشبكة بأسماء وصفية وبناء التخطيط بشكل مرئي جداً — وكأنك ترسم تخطيط الصفحة مباشرة في CSS!

4.1

طريقة الاستخدام

CSS 
/* الخطوة 1: تعريف مناطق الشبكة في الحاوية */
.page {
  display: grid;
  grid-template-columns: 260px 1fr;
  grid-template-rows: 70px 1fr 60px;
  min-height: 100vh;
  gap: 0;

  grid-template-areas:
    "header  header"
    "sidebar main  "
    "footer  footer";
}

/* الخطوة 2: تعيين كل عنصر لمنطقته */
header  { grid-area: header;  }
.sidebar { grid-area: sidebar; }
main    { grid-area: main;    }
footer  { grid-area: footer;  }
🗺️ تصور grid-template-areas
  grid-template-areas:
    "header  header"    ← الصف 1: header يمتد عمودين
    "sidebar main  "    ← الصف 2: sidebar | main
    "footer  footer";   ← الصف 3: footer يمتد عمودين
  النتيجة البصرية:
  ┌──────────────────────────────────┐
  │            header                │
  ├──────────────┬───────────────────┤
  │   sidebar    │       main        │
  │              │                   │
  ├──────────────┴───────────────────┤
  │            footer                │
  └──────────────────────────────────┘
4.2

مثال: تخطيط ثلاثة أعمدة مع header وfooter

CSS 
.page-3col {
  display: grid;
  grid-template-columns: 220px 1fr 220px;
  grid-template-rows: 70px 1fr auto 60px;
  min-height: 100vh;

  grid-template-areas:
    "header  header  header "
    "sidebar main    aside  "
    "sidebar content aside  "
    "footer  footer  footer ";

header  { grid-area: header;  background: #1A3C5E; color: white; }
.sidebar { grid-area: sidebar; background: #EBF3FB; }
main    { grid-area: main;    padding: 30px; }
.content { grid-area: content; padding: 30px; }
aside   { grid-area: aside;   background: #F5EEFF; }
footer  { grid-area: footer;  background: #1A1A2E; color: white; }

الجزء الخامس: المسافات والمحاذاة في Grid

5.1

المسافات (Gap)

CSS 
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);

  /* مسافة موحدة بين جميع الخلايا */
  gap: 20px;

  /* مسافة مختلفة: أفقية وعمودية */
  gap: 16px 24px;    /* صفوف: 16px | أعمدة: 24px */

  /* تفصيلي */
  row-gap: 16px;
  column-gap: 24px;
}
5.2

محاذاة الشبكة كلها (justify-content & align-content)

CSS 
/* محاذاة الشبكة كلها داخل الحاوية */
.container {
  display: grid;
  grid-template-columns: repeat(3, 200px); /* أعمدة ثابتة تاركة مساحة */
  width: 100%;

  /* توزيع الشبكة أفقياً */
  justify-content: center;        /* توسيط الشبكة */
  justify-content: space-between; /* توزيع موزون */
  justify-content: start;         /* الافتراضي */
  justify-content: end;

  /* توزيع الشبكة عمودياً */
  align-content: center;
  align-content: space-between;
  align-content: start;
}
5.3

محاذاة العناصر داخل خلاياها

CSS 
/* محاذاة جميع العناصر داخل خلاياها */
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);

  /* أفقياً داخل الخلية */
  justify-items: start;    /* محاذاة يسار */
  justify-items: end;      /* محاذاة يمين */
  justify-items: center;   /* توسيط أفقي */
  justify-items: stretch;  /* الافتراضي: يملأ الخلية */

  /* عمودياً داخل الخلية */
  align-items: start;    /* أعلى */
  align-items: end;      /* أسفل */
  align-items: center;   /* توسيط عمودي */
  align-items: stretch;  /* الافتراضي */
}

/* محاذاة عنصر واحد بشكل مستقل */
.special-item {
  justify-self: center;   /* توسيط أفقي لهذا العنصر فقط */
  align-self: end;        /* في أسفل خليته فقط */
}

/* توسيط عنصر كاملاً داخل خليته */
.centered-item {
  justify-self: center;
  align-self: center;
}
5.4

جدول مرجع: place-content وplace-items وplace-self

الاختصارما يُختصره
place-content: centeralign-content: center + justify-content: center
place-items: centeralign-items: center + justify-items: center
place-self: centeralign-self: center + justify-self: center
place-content: end startalign-content: end + justify-content: start

الجزء التاسع: تمارين الجلسة الخامسة

✏️ تمرين 1 — الأساسيات15 دقيقة

أنشئ شبكة بسيطة وطبّق ما يلي:

  • شبكة 4 أعمدة بوحدة fr متساوية مع 9 عناصر ملونة
  • شبكة بأعمدة: 200px 1fr 2fr 1fr مع gap: 20px
  • العنصر الأول يمتد عبر أول عمودين (grid-column: span 2)
  • العنصر الأخير يمتد عبر آخر عمودين بالقيمة السالبة
  • جرّب: repeat(auto-fit, minmax(150px, 1fr)) وتغيير حجم النافذة

✏️ تمرين 2 — grid-template-areas25 دقيقة

أنشئ تخطيط صفحة كاملة باستخدام grid-template-areas:

  • header: شريط أزرق داكن يمتد عرض الصفحة كاملاً
  • nav: شريط أفقي بروابط تحت الـ header
  • sidebar: شريط جانبي أيمن عرضه 240px
  • main: قسم المحتوى الرئيسي يأخذ الباقي
  • ads: شريط جانبي أيسر عرضه 180px
  • footer: تذييل يمتد عرض الصفحة كاملاً
  • ضع محتوى وهمياً في كل منطقة مع لون خلفية مختلف

✏️ تمرين 3 — تخطيط المجلة30 دقيقة

أنشئ صفحة فئة بشبكة مقالات تحتوي على:

  • مقال رئيسي يمتد عبر عمودين وصفين (featured)
  • مقالان جانبيان بجانب المقال الرئيسي
  • صف من 4 مقالات صغيرة تحت الجميع
  • hover effect على كل مقال: ارتفاع + تغيير ظل
  • استخدم grid-auto-rows: minmax(200px, auto) للصفوف
  • أضف صور placeholder مع object-fit: cover

الموضوعأبرز ما تعلمناه
Grid vs FlexboxGrid ثنائي الأبعاد للتخطيط الكامل | Flexbox أحادي البُعد للعناصر الداخلية
تفعيل Griddisplay: grid — الأبناء المباشرون يصبحون Grid Items
تعريف الأعمدةgrid-template-columns: px | fr | % | repeat() | minmax()
وحدة frكسر من المساحة المتبقية — أقوى وحدة في Grid
repeat()repeat(3, 1fr) اختصار لـ 1fr 1fr 1fr
minmax()minmax(280px, 1fr) — حد أدنى وأقصى للعمود
الوصفة الذهبيةrepeat(auto-fit, minmax(MIN, 1fr)) — شبكة متجاوبة بلا Media Queries
وضع العناصرgrid-column: start / end | span n | 1 / -1 للعرض الكامل
grid-template-areasتسمية المناطق وبناء التخطيط بشكل مرئي في CSS
المسافاتgap | row-gap | column-gap
المحاذاةjustify-items, align-items, justify-self, align-self — نفس مفهوم Flexbox
دمج Grid + FlexGrid للتخطيط الخارجي | Flexbox للمحتوى الداخلي لكل قسم
