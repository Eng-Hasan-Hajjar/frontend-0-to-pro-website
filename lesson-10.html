الجزء الأول: ما هو DOM؟

1.1

تعريف DOM

DOM (Document Object Model) هو واجهة برمجية تمثّل صفحة HTML كشجرة من العقد (Nodes). كل عنصر HTML يُصبح «عقدة» في هذه الشجرة يمكن لـ JavaScript الوصول إليها وتعديلها.

🌳 شجرة DOM لصفحة HTML بسيطة
  document
  └── html
      ├── head
      │   ├── title → 'أكاديمية التقنية'
      │   └── link (stylesheet)
      └── body
          ├── header
          │   ├── h1 → 'مرحباً'
          │   └── nav
          │       ├── a → 'الرئيسية'
          │       └── a → 'من نحن'
          ├── main
          │   ├── section#hero
          │   │   └── p → 'نص البطل'
          │   └── section#courses
          └── footer
              └── p → '© 2025'

كائن document

document هو نقطة الدخول الرئيسية للـ DOM — يمثّل الصفحة كاملة:

JavaScript 
// document يمثّل الصفحة بأكملها
console.log(document);              // الصفحة كلها
console.log(document.title);        // عنوان الصفحة
console.log(document.URL);          // رابط الصفحة الحالية
console.log(document.body);         // عنصر <body>
console.log(document.head);         // عنصر <head>
console.log(document.documentElement); // عنصر <html>

// تعديل عنوان الصفحة ديناميكياً
document.title = 'صفحة جديدة';

// تغيير لون خلفية الصفحة مباشرة
document.body.style.backgroundColor = '#F0F6FF';

الجزء الثاني: تحديد العناصر (Selecting Elements)

قبل أن تُعدّل أي عنصر، يجب أن تُحدّده أولاً. JavaScript توفر عدة طرق لذلك:

2.1

querySelector وquerySelectorAll (الأحدث والأقوى ✅)

JavaScript 
// querySelector — يُعيد أول عنصر مطابق
const title    = document.querySelector('h1');
const btn      = document.querySelector('.btn-primary');
const mainNav  = document.querySelector('#main-nav');
const firstLi  = document.querySelector('ul li');
const emailInp = document.querySelector('input[type="email"]');
const active   = document.querySelector('.nav-link.active');

// إذا لم يُوجد العنصر تُعاد null
const notFound = document.querySelector('.nothing');
console.log(notFound); // null

// querySelectorAll — يُعيد NodeList بجميع المطابقات
const allBtns    = document.querySelectorAll('.btn');
const allInputs  = document.querySelectorAll('input');
const navLinks   = document.querySelectorAll('.nav-link');
const evenRows   = document.querySelectorAll('tr:nth-child(even)');

// التكرار على NodeList
navLinks.forEach(link => {
  console.log(link.textContent);
});

// تحويل NodeList لـ Array للمزيد من المرونة
const linksArray = Array.from(navLinks);
const filtered   = linksArray.filter(l => l.classList.contains('active'));

// البحث داخل عنصر محدد (وليس الصفحة كلها)
const card     = document.querySelector('.card');
const cardTitle = card.querySelector('h3');  // h3 داخل .card فقط
2.2

الطرق الكلاسيكية

JavaScript 
// getElementById — بالمعرّف id
const header = document.getElementById('main-header');
// يُعيد العنصر مباشرة أو null

// getElementsByClassName — بالـ class
const cards = document.getElementsByClassName('card');
// يُعيد HTMLCollection (حي — يتحدث تلقائياً)

// getElementsByTagName — باسم الوسم
const allDivs = document.getElementsByTagName('div');
const allImgs = document.getElementsByTagName('img');

// getElementsByName — بخاصية name (للنماذج)
const radioGroup = document.getElementsByName('gender');
الطريقةما تُعيده وملاحظات
querySelector()أول عنصر مطابق | null إذا لم يوجد | يقبل CSS Selectors ✅
querySelectorAll()NodeList بجميع المطابقات | يقبل CSS Selectors ✅
getElementById()عنصر واحد بـ id | أسرع لأن id فريد
getElementsByClassName()HTMLCollection حية | تتحدث عند تغيير DOM
getElementsByTagName()HTMLCollection بجميع عناصر الوسم
💡 أيهما تستخدم؟

querySelector وquerySelectorAll في 95% من الحالات — أقوى وأكثر مرونة

getElementById عندما تحتاج أداءً عالياً لعنصر محدد بـ id

getElementsByClassName وgetElementsByTagName للكود القديم فقط

الجزء الثالث: قراءة المحتوى وتعديله

3.1

textContent وinnerHTML وinnerText

JavaScript 
const el = document.querySelector('#message');

// ══ textContent — النص الخام (بدون HTML) ✅ ══
console.log(el.textContent);         // النص فقط
el.textContent = 'نص جديد آمن';      // يُعامل كنص (لا يُفسَّر كـ HTML)
el.textContent = '<b>لا يُعرض</b>'; // يُعرض حرفياً كنص

// ══ innerHTML — النص مع HTML ══
console.log(el.innerHTML);           // النص مع وسوم HTML
el.innerHTML = '<strong>نص غامق</strong>'; // يُفسَّر كـ HTML ✅
el.innerHTML = '';                   // مسح كل المحتوى الداخلي

// ⚠️ تحذير XSS — لا تضع مُدخَل المستخدم مباشرة في innerHTML!
// el.innerHTML = userInput; // ❌ خطر أمني!
// استخدم textContent للمُدخَلات دائماً

// ══ outerHTML — العنصر كاملاً مع وسوم الإغلاق ══
console.log(el.outerHTML);
// '<div id="message">النص</div>'

// ══ innerText — مثل textContent لكن يراعي التنسيق المرئي ══
// (لا يُعيد النصوص المخفية بـ CSS)
console.log(el.innerText);
3.2

الخصائص (Attributes)

JavaScript 
const link = document.querySelector('a.hero-link');
const img  = document.querySelector('img.avatar');
const inp  = document.querySelector('#email-input');

// ══ قراءة خاصية ══
console.log(link.getAttribute('href'));   // '/about'
console.log(img.getAttribute('alt'));     // 'صورة المستخدم'
console.log(inp.getAttribute('type'));    // 'email'

// ══ تعديل خاصية ══
link.setAttribute('href', '/contact');
img.setAttribute('src', 'new-photo.jpg');
img.setAttribute('alt', 'صورة جديدة');

// ══ حذف خاصية ══
link.removeAttribute('target');

// ══ التحقق من وجود خاصية ══
console.log(inp.hasAttribute('required')); // true أو false

// ══ الخصائص المباشرة (Dot Notation) — أسرع ✅ ══
console.log(link.href);        // رابط كامل
console.log(img.src);          // مسار كامل
console.log(img.alt);          // النص البديل
console.log(inp.value);        // قيمة حقل الإدخال
console.log(inp.type);         // نوع الحقل
console.log(inp.disabled);     // boolean
console.log(inp.checked);      // للـ checkbox/radio

// تعديل مباشر
inp.value    = 'ahmed@example.com';
inp.disabled = true;
inp.placeholder = 'أدخل بريدك';

// dataset — الخصائص المخصصة (data-*)
// <div data-user-id='42' data-role='admin'>
const userCard = document.querySelector('.user-card');
console.log(userCard.dataset.userId); // '42'
console.log(userCard.dataset.role);   // 'admin'
userCard.dataset.status = 'active';   // يُضيف data-status='active'

4.1

خاصية style المباشرة

JavaScript 
const box = document.querySelector('.box');

// قراءة
console.log(box.style.backgroundColor); // '' فارغ إذا لم يُحدَّد inline
console.log(box.style.width);

// تعديل خاصية واحدة
box.style.backgroundColor = '#2E75B6';
box.style.color = 'white';
box.style.width = '300px';
box.style.padding = '20px';
box.style.borderRadius = '12px';  // camelCase بدلاً من border-radius
box.style.fontSize = '1.2rem';
box.style.display = 'none';       // إخفاء
box.style.display = '';           // إعادة لافتراضي (إظهار)

// تعديل متعدد دفعة واحدة بـ cssText
box.style.cssText = `
  background-color: #1A3C5E;
  color: white;
  padding: 24px;
  border-radius: 16px;
  font-size: 1rem;
`;

// قراءة القيمة المحسوبة الفعلية (بما فيها CSS الخارجي)
const computed = getComputedStyle(box);
console.log(computed.backgroundColor); // 'rgb(26, 60, 94)'
console.log(computed.fontSize);        // '16px'
console.log(computed.display);         // 'block'
4.2

classList — الطريقة الاحترافية ✅

أفضل بكثير من style المباشرة — تعمل مع CSS المُعرَّف مسبقاً:

JavaScript 
const card = document.querySelector('.card');

// ══ add — إضافة class ══
card.classList.add('card--active');
card.classList.add('card--highlighted', 'animate-in'); // متعدد

// ══ remove — حذف class ══
card.classList.remove('card--loading');
card.classList.remove('hidden', 'disabled');

// ══ toggle — إضافة إن لم يكن موجود، حذف إن كان موجوداً ✅ ══
card.classList.toggle('card--expanded');
card.classList.toggle('dark-mode');

// toggle مع قيمة (force)
card.classList.toggle('active', true);  // يُضيف دائماً
card.classList.toggle('active', false); // يحذف دائماً

// ══ contains — التحقق من وجود class ══
if (card.classList.contains('card--featured')) {
  console.log('هذه بطاقة مميزة');
}

// ══ replace — استبدال class بآخر ══
card.classList.replace('card--loading', 'card--loaded');

// ══ قراءة جميع الـ classes ══
console.log(card.className);              // نص
console.log(card.classList.toString());   // نص
console.log([...card.classList]);         // مصفوفة
console.log(card.classList.length);       // العدد

// مثال عملي: Dark Mode Toggle
const toggleBtn = document.querySelector('#theme-toggle');
toggleBtn.addEventListener('click', () => {
  document.body.classList.toggle('dark-mode');
  const isDark = document.body.classList.contains('dark-mode');
  toggleBtn.textContent = isDark ? '☀️ فاتح' : '🌙 داكن';
});

const newCard = document.createElement('article');
const newBtn  = document.createElement('button');
const newImg  = document.createElement('img');
const newPara = document.createElement('p');

// إعداد العنصر الجديد
newCard.className = 'product-card';
newCard.id = 'card-42';
newCard.setAttribute('data-product-id', '42');

newBtn.textContent = 'أضف للسلة 🛒';
newBtn.className = 'btn btn--primary';
newBtn.type = 'button';

newImg.src = 'product.jpg';
newImg.alt = 'صورة المنتج';
newImg.loading = 'lazy'; // تحميل كسول

newPara.innerHTML = `السعر: <strong>99 ريال</strong>`;

// ══ إضافة العنصر للصفحة ══
const container = document.querySelector('.products-grid');

// appendChild — إضافة في نهاية الأب

// prepend — إضافة في بداية الأب
container.prepend(newCard);

// insertBefore — إضافة قبل عنصر محدد
const thirdCard = container.querySelector(':nth-child(3)');
container.insertBefore(newCard, thirdCard);

// insertAdjacentHTML — أسرع للمحتوى الجاهز ✅
container.insertAdjacentHTML('beforeend',

// 'afterbegin'  — داخل العنصر في البداية
// 'beforeend'   — داخل العنصر في النهاية ✅
// 'afterend'    — بعد العنصر

}

الجزء السادس: التنقل في شجرة DOM (Traversal)

6.1

خصائص التنقل

JavaScript 
const item = document.querySelector('.list-item');

// ══ الأب (Parent) ══
console.log(item.parentElement);      // العنصر الأب (أفضل من parentNode)
console.log(item.parentElement.parentElement); // الجد
console.log(item.closest('ul'));       // يبحث للأعلى عن المطابق ✅
console.log(item.closest('.container')); // مفيد جداً

// ══ الأبناء (Children) ══
const list = document.querySelector('ul');
console.log(list.children);           // HTMLCollection بالأبناء المباشرين
console.log(list.childElementCount);  // عدد الأبناء
console.log(list.firstElementChild);  // أول ابن
console.log(list.lastElementChild);   // آخر ابن
console.log([...list.children]);      // مصفوفة من الأبناء

// ══ الأشقاء (Siblings) ══
console.log(item.nextElementSibling);     // الشقيق التالي
console.log(item.previousElementSibling); // الشقيق السابق

// ══ مثال: تلوين كل عنصر قائمة بالتناوب ══
const items = document.querySelectorAll('li');
items.forEach((item, index) => {
  item.style.backgroundColor = index % 2 === 0 ? '#F0F6FF' : 'white';
});
6.2

closest() لإيجاد الأب المطابق

JavaScript 
// closest() — يبحث للأعلى في الشجرة حتى يجد مطابقاً
// مفيد جداً مع Event Delegation

// <div class='card'>
//   <h3>عنوان</h3>
//   <button class='delete-btn'>حذف</button>
// </div>

document.addEventListener('click', (e) => {
  if (e.target.classList.contains('delete-btn')) {
    const card = e.target.closest('.card'); // يُعيد .card الأب
    if (card) card.remove();
  }
});

// مثال: تحديد صف الجدول من زر داخله
document.querySelector('table').addEventListener('click', (e) => {
  const btn = e.target.closest('[data-action]');
  if (!btn) return;

  const row = btn.closest('tr');
  const action = btn.dataset.action;

  if (action === 'delete') {
    row.remove();
  } else if (action === 'edit') {
    row.classList.add('editing');
  }
});

الجزء السابع: الأحداث (Events)

الأحداث هي طريقة JavaScript للاستجابة لتفاعلات المستخدم: النقر، الكتابة، التمرير، تحريك الماوس، وغيرها.

7.1

addEventListener (الطريقة الاحترافية ✅)

JavaScript 
const btn = document.querySelector('#submit-btn');

// الصيغة: element.addEventListener(event, handler, options)
btn.addEventListener('click', function(event) {
  console.log('تم النقر!');
  console.log(event.target);    // العنصر الذي نُقر عليه
});

// مع Arrow Function
btn.addEventListener('click', (e) => {
  console.log('الزر نُقر:', e.target.textContent);
});

// إضافة مستمعَيْن لنفس الحدث — كلاهما يعمل
btn.addEventListener('click', handleFirstClick);
btn.addEventListener('click', handleSecondClick);

// إزالة مستمع الحدث
btn.removeEventListener('click', handleFirstClick);

// once: true — يُنفَّذ مرة واحدة فقط ✅
btn.addEventListener('click', () => {
  console.log('هذا يظهر مرة واحدة فقط!');
}, { once: true });

Event Object (كائن الحدث)

JavaScript 
document.querySelector('form').addEventListener('submit', (event) => {
  // ══ منع السلوك الافتراضي ══
  event.preventDefault(); // منع إرسال النموذج وإعادة تحميل الصفحة

  // ══ خصائص Event Object ══
  console.log(event.type);           // 'submit'
  console.log(event.target);         // النموذج نفسه
  console.log(event.currentTarget);  // العنصر الذي يستمع
  console.log(event.timeStamp);      // وقت الحدث

document.querySelector('.canvas').addEventListener('mousemove', (e) => {

Event Delegation (تفويض الأحداث) ✅

بدلاً من إضافة مستمع لكل عنصر، نضيف مستمعاً واحداً للأب — يُوفّر الأداء ويعمل مع العناصر الديناميكية:

JavaScript 
// ❌ طريقة سيئة — مستمع لكل زر
document.querySelectorAll('.delete-btn').forEach(btn => {
  btn.addEventListener('click', () => { /* ... */ });
});
// مشكلة: لا يعمل مع الأزرار المضافة لاحقاً!

// ✅ Event Delegation — مستمع واحد على الأب
const taskList = document.querySelector('#task-list');

taskList.addEventListener('click', (e) => {
  // e.target هو العنصر الذي نُقر عليه فعلاً

  // التحقق من نوع الزر المنقور
  if (e.target.classList.contains('btn-delete')) {
    const task = e.target.closest('.task-item');
    task.remove();
  }

  if (e.target.classList.contains('btn-complete')) {
    const task = e.target.closest('.task-item');
    task.classList.toggle('task--completed');
  }

  if (e.target.classList.contains('btn-edit')) {
    const task = e.target.closest('.task-item');
    const title = task.querySelector('.task-title');
    title.contentEditable = 'true';
    title.focus();
  }
});

// الآن حتى المهام المضافة لاحقاً لها نفس السلوك!

ملف app.js

JavaScript 
/* ═══════════════════════════════════════════════ */
/* To-Do List App — DOM Session 10                */
/* ═══════════════════════════════════════════════ */

// ══ 1. تحديد العناصر ══
const addForm      = document.getElementById('add-form');
const taskInput    = document.getElementById('task-input');
const prioritySel  = document.getElementById('priority-select');
const taskList     = document.getElementById('task-list');
const emptyState   = document.getElementById('empty-state');
const totalCount   = document.getElementById('total-count');
const doneCount    = document.getElementById('done-count');
const clearDoneBtn = document.getElementById('clear-done');
const filterBtns   = document.querySelectorAll('.filter-btn');

// ══ 2. البيانات ══
let tasks = [];
let nextId = 1;
let currentFilter = 'all';

// ══ 3. إنشاء عنصر مهمة ══
function createTaskElement(task) {
  const li = document.createElement('li');
  li.className = `task-item task-item--${task.priority}
    ${task.done ? 'task-item--done' : ''}`.trim();
  li.dataset.taskId = task.id;

  li.innerHTML = `
    <label class="task-check">
      <input type="checkbox" ${task.done ? 'checked' : ''}
        data-action="toggle" />
      <span class="task-title">${escapeHtml(task.title)}</span>
    </label>
    <div class="task-actions">
      <span class="task-priority priority--${task.priority}">
        ${getPriorityLabel(task.priority)}
      </span>
      <button class="btn-icon" data-action="edit" title="تعديل">✏️</button>
      <button class="btn-icon btn-icon--danger" data-action="delete"
        title="حذف">🗑️</button>
    </div>
  `;

  return li;
}

// ══ 4. دالة عرض المهام ══
function renderTasks() {
  // تصفية المهام
  const filtered = tasks.filter(task => {
    if (currentFilter === 'active') return !task.done;
    if (currentFilter === 'done')   return task.done;
    return true; // 'all'
  });

  // مسح القائمة
  taskList.innerHTML = '';

  // حالة القائمة الفارغة
  if (filtered.length === 0) {
    const msg = currentFilter === 'done'
      ? '😊 لا توجد مهام مكتملة'
      : '🎉 لا توجد مهام!';
    taskList.innerHTML =
      `<li class="empty-state">${msg}</li>`;
    updateStats();
    return;
  }

  // رسم المهام
  filtered.forEach(task => {
    taskList.appendChild(createTaskElement(task));
  });

  updateStats();
}

// ══ 5. تحديث الإحصائيات ══
function updateStats() {
  const total = tasks.length;
  const done  = tasks.filter(t => t.done).length;
  totalCount.textContent = `${total} مهمة`;
  doneCount.textContent  = `${done} مكتملة`;
}

// ══ 6. إضافة مهمة ══
function addTask(title, priority) {
  const task = {
    id:       nextId++,
    title:    title.trim(),
    priority: priority,
    done:     false,
    createdAt: new Date().toLocaleDateString('ar-SA'),
  };
  tasks.unshift(task); // إضافة للبداية
  renderTasks();
}

// ══ 7. حذف مهمة ══
function deleteTask(id) {
  tasks = tasks.filter(t => t.id !== id);
  renderTasks();
}

// ══ 8. تبديل حالة إنجاز المهمة ══
function toggleTask(id) {
  const task = tasks.find(t => t.id === id);
  if (task) {
    task.done = !task.done;
    renderTasks();
  }
}

// ══ 9. تعديل مهمة ══
function editTask(id, li) {
  const task  = tasks.find(t => t.id === id);
  const title = li.querySelector('.task-title');
  const newText = prompt('تعديل المهمة:', task.title);
  if (newText && newText.trim() !== '') {
    task.title = newText.trim();
    renderTasks();
  }
}

// ══ 10. مساعدات ══
function escapeHtml(text) {
  const div = document.createElement('div');
  div.textContent = text;
  return div.innerHTML;
}

function getPriorityLabel(p) {
  return p === 'high' ? '🔴 عالية'
       : p === 'low'  ? '🟢 منخفضة'
       :                '🟡 عادية';
}

// ══ 11. مستمعات الأحداث ══

// إضافة مهمة
addForm.addEventListener('submit', (e) => {
  e.preventDefault();
  const title = taskInput.value.trim();
  if (!title) return;
  addTask(title, prioritySel.value);
  taskInput.value = '';
  taskInput.focus();
});

// Event Delegation — إجراءات المهام
taskList.addEventListener('click', (e) => {
  const action = e.target.closest('[data-action]')?.dataset.action;
  if (!action) return;

  const li   = e.target.closest('.task-item');
  const id   = Number(li.dataset.taskId);

  switch (action) {
    case 'toggle': toggleTask(id); break;
    case 'delete': deleteTask(id); break;
    case 'edit':   editTask(id, li); break;
  }
});

// أزرار التصفية
document.querySelector('.filters').addEventListener('click', (e) => {
  const btn = e.target.closest('.filter-btn');
  if (!btn) return;

  filterBtns.forEach(b => b.classList.remove('active'));
  btn.classList.add('active');
  currentFilter = btn.dataset.filter;
  renderTasks();
});

// حذف المهام المكتملة
clearDoneBtn.addEventListener('click', () => {
  if (!confirm('حذف جميع المهام المكتملة؟')) return;
  tasks = tasks.filter(t => !t.done);
  renderTasks();
});

// اختصار Enter لإضافة وEscape للإغلاق
taskInput.addEventListener('keydown', (e) => {
  if (e.key === 'Escape') taskInput.blur();
});

// ══ 12. تشغيل أولي ══
renderTasks();
taskInput.focus();

الجزء التاسع: تمارين الجلسة العاشرة

✏️ تمرين 1 — تحديد العناصر وتعديلها15 دقيقة

في Console المتصفح على أي موقع تختاره:

  • حدّد عنوان الصفحة وغيّر نصه
  • أضف class='highlight' لجميع الفقرات بـ querySelectorAll + forEach
  • حدّد أول رابط وغيّر href الخاص به
  • أضف زر HTML جديد في نهاية body بـ insertAdjacentHTML
  • غيّر لون خلفية body عند النقر على الزر باستخدام addEventListener

✏️ تمرين 2 — إنشاء وحذف العناصر ديناميكياً20 دقيقة

أنشئ صفحة بـ HTML وJS تحتوي على:

  • حقل إدخال اسم المهارة + حقل نسبة (0-100) + زر إضافة
  • عند النقر: أنشئ div جديد يحتوي على اسم المهارة وشريط تقدم بالنسبة المحددة
  • كل شريط تقدم له لون مختلف حسب النسبة: أحمر < 50, أصفر < 75, أخضر ≥ 75
  • أضف زر حذف لكل مهارة يحذفها عند النقر
  • عدّاد يُظهر عدد المهارات المضافة

✏️ تمرين 3 — Event Delegation وأحداث متقدمة25 دقيقة

أنشئ جدولاً تفاعلياً لقائمة موظفين:

  • جدول يحتوي 5 موظفين: الاسم، القسم، الراتب، الحالة (نشط/غير نشط)
  • زر 'حذف' في كل صف يحذف الصف بـ Event Delegation على الجدول
  • النقر على الصف يُبرزه ويُلغي تبرز الصفوف الأخرى
  • حقل بحث يُصفّي الصفوف المرئية بـ textContent.includes() عند كل input
  • زر 'تصدير' يُطبع أسماء الموظفين في console
