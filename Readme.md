
# Front-End Master Book — مسار JavaScript

موقع تفاعلي لتعليم JavaScript باللغة العربية، بديل عن عروض PowerPoint — إعداد **م. حسن حجار**.

## التشغيل

لا حاجة لأي تثبيت. افتح `index.html` مباشرة في أي متصفح حديث (Chrome, Edge, Firefox, Safari).

> ملاحظة: بعض المتصفحات تمنع `fetch`/الخطوط عند الفتح المباشر من القرص (`file://`). إن واجهت مشكلة في تحميل الخطوط، شغّل خادماً محلياً بسيطاً:
>
> ```bash
> npx serve .
> # أو
> python3 -m http.server 8000
> ```

## هيكل الملفات

```
index.html        الصفحة الرئيسية (البطل + شبكة الجلسات + عن المدرب)
style.css         كل التنسيقات (فاتح/داكن، الأكواد، البطاقات...)
script.js         كل التفاعلات (الوضع الليلي، الشريط الجانبي، تشغيل الأكواد، حفظ التقدّم)
lesson-09.html    الجلسة 09 — مقدمة إلى JavaScript
lesson-10.html    الجلسة 10 — JavaScript والـ DOM
lesson-11.html    الجلسة 11 — المصفوفات والكائنات المتقدمة
lesson-12.html    الجلسة 12 — البرمجة غير المتزامنة
lesson-13.html    الجلسة 13 — التخزين المحلي في المتصفح
lesson-14.html    الجلسة 14 — البرمجة الكائنية OOP
```

المسار الست جلسات مكتمل بالكامل ومترابط بأزرار "السابق/التالي".

## كيف يعمل الموقع

- **الوضع الليلي/النهاري**: زر 🌙/☀️ في الشريط العلوي، يُحفَظ الاختيار في `localStorage`.
- **تتبّع التقدّم**: زر "أكملت هذا الدرس" أسفل كل جلسة يُسجَّل محلياً في متصفح المتعلم (لا خادم، لا قاعدة بيانات).
- **تشغيل الأكواد**: زر "▶ جرّب الكود" ينفّذ الكود داخل بيئة معزولة (باستخدام `new Function`) ويعرض ناتج `console.log/warn/error` في صندوق تحت الكود — دون أي اتصال بالإنترنت.
- **تلوين الأكواد (Syntax Highlighting)**: مُنفَّذ يدوياً بـ JavaScript خالص (بدون Prism.js أو أي مكتبة خارجية) في `script.js`.
- **قائمة الجلسات** (الصفحة الرئيسية + الشريط الجانبي) تُبنى تلقائياً من مصفوفة `SESSIONS` في أعلى `script.js` — لا حاجة لتعديل HTML يدوياً عند إضافة جلسة جديدة.

## إضافة جلسة جديدة لاحقاً (مثلاً الجلسة 15)

الجلسات الست الحالية بُنيت آلياً من ملفات Word الأصلية عبر أداة تحويل (وليست مكتوبة يدوياً بالكامل) لضمان التطابق التام في الأسلوب. إن رغبت بإضافة جلسات مستقبلية بنفس الطريقة، شارك ملف Word للجلسة الجديدة وسيُعاد استخدام نفس الأداة لإنشائها بنفس القالب البصري والتفاعلي — وإلا يمكن إنشاء `lesson-15.html` يدوياً بنفس بنية الملفات الحالية (رأس الجلسة → بطاقة الأهداف → الأجزاء → الواجب → الملخص → شارة الختام)، مع تحديث مصفوفة `SESSIONS` في `script.js` وتحديث روابط "السابق/التالي" في الجلسة 14 والجلسة الجديدة.

## التخصيص السريع

- **الألوان**: كل الألوان متغيرات CSS في أعلى `style.css` (`:root` للوضع الفاتح، `[data-theme="dark"]` للداكن) — غيّر `--primary` و`--accent` لتغيير الهوية اللونية بالكامل.
- **الخطوط**: Cairo (العناوين) وIBM Plex Sans Arabic (النصوص) عبر Google Fonts، وFira Code للأكواد — روابطها في `<head>` كل صفحة.
- **اسم الموقع / المدرّب**: غيّر النص داخل وسم `.brand` في `index.html` وقالب الجلسات.
